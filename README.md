# 🔍 Search Previewer  

> A fast, minimal, and smart search experience with **live preview + intent detection**

🌐 **Live Demo:** https://shankar2006458.github.io/Search-Previewer/

---

## ✨ What is this?

**Search Previewer** is a lightweight web app that improves the traditional search experience.  
Instead of blindly clicking links, you get a **real-time preview** of results + **intent detection** before visiting any website.

Built for speed, clarity, and a smoother browsing experience.

---

## 🚀 Features  

### 🔎 Smart Search Experience
- Instant Google-based search  
- Clean and distraction-free UI  
- Fast response, no lag  

### 👀 Live Preview
- Preview websites **before opening them**  
- Saves time and unnecessary clicks  
- Shows multiple preview links (5+ results)  

### 🧠 Intent Detection
- Detects user intent from query:
  - 🛒 Shopping (`buy`, `price`)
  - 📘 Informational (`how`, `what`, `guide`)
  - 🎥 Video (`watch`, `video`)
  - 🌐 General (default)  
- Displays intent badge dynamically  

### ⚡ Smooth UX
- Top progress loading bar while typing  
- Skeleton loading (no flicker)  
- Clean transitions  

### 🎯 Extra Touch
- No history clutter  
- Minimal UI focused on usability  
- Fully responsive (mobile + desktop)  

---

## 🛠️ Tech Stack  

- HTML5  
- CSS3 / Tailwind CSS  
- Vanilla JavaScript  
- Google Search Redirect Logic  


## 📦 How it Works  

1. User types a query  
2. App detects **intent** using keyword logic  
3. Generates Google search links  
4. Displays:
   - Preview links  
   - Intent badge  
   - Live loading feedback
